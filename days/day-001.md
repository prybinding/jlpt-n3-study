  <h2>오늘의 목표 (추천 60~90분)</h2>
  <ul>
    <li>히라가나/가타카나가 “뭔지” 이해 + 학습 루틴 세팅</li>
    <li>기본 인사 10개 암기 (표기/발음/뜻)</li>
    <li>초간단 문장 패턴 2개 맛보기</li>
  </ul>

  <h2>0) 루틴(중요)</h2>
  <ul>
    <li><b>입으로 소리 내기</b>: 일본어는 “문자”보다 “소리(모라)”가 핵심.</li>
    <li><b>매일 복습</b>: D+1 / D+3 / D+7 / D+14 형태로 재노출.</li>
    <li><b>기억법</b>: 무조건 외우기보다, “연상 고리”를 만들어서 붙잡기.</li>
  </ul>

  <h2>1) 일본어 문자 개요 (5분)</h2>
  <ul>
    <li><b>히라가나(ひらがな)</b>: 일본어 기본 소리(가나)를 적는 문자. (평소 문장에 주로 사용)</li>
    <li><b>가타카나(カタカナ)</b>: 외래어/강조/의성어 등에 자주 사용</li>
    <li><b>한자(漢字)</b>: 의미 단위. N3는 한자도 꽤 나옴 (하지만 1일차엔 개념만)</li>
  </ul>

  <h2>2) 발음 규칙 맛보기 (10분)</h2>
  <ul>
    <li><b>장음(ー / おう / えい)</b>: 길게 늘려 읽는 소리. 의미가 바뀔 수 있음.</li>
    <li><b>촉음(っ)</b>: 잠깐 막았다가 다음 자음을 강하게. (예: きって = 킷테)</li>
    <li><b>탁음/반탁음(゛/゜)</b>: か→が / は→ぱ 같은 소리 변화.</li>
  </ul>

  <h2>3) 오늘의 핵심 표현 10개 (표기 / 읽기 / 뜻 / 기억법)</h2>
  <ol>
    <li><b>こんにちは</b> / (kon-ni-chi-wa) / 안녕하세요(낮 인사)
      <ul><li>기억법: “곤니치와”를 한 덩어리로 통째로.</li></ul>
    </li>
    <li><b>おはよう</b> / (o-ha-yo-u) / 안녕(아침)
      <ul><li>기억법: “오~ 하요~” 아침에 길게.</li></ul>
    </li>
    <li><b>こんばんは</b> / (kon-ban-wa) / 안녕하세요(저녁 인사)</li>
    <li><b>ありがとう</b> / (a-ri-ga-to-u) / 고마워
      <ul><li>기억법: “아리가또”는 자주 들리니까 귀로 익히기.</li></ul>
    </li>
    <li><b>すみません</b> / (su-mi-ma-sen) / 죄송합니다/실례합니다/저기요
      <ul><li>기억법: 상황万能(만능) 표현. ‘스미마셍’ 하나로 다 열림.</li></ul>
    </li>
    <li><b>はい</b> / (hai) / 네</li>
    <li><b>いいえ</b> / (i-i-e) / 아니요</li>
    <li><b>お願いします</b> / (o-ne-ga-i-shi-ma-su) / 부탁합니다(주세요/부탁)
      <ul><li>기억법: 주문할 때 마지막에 붙이면 대부분 통함.</li></ul>
    </li>
    <li><b>わかりました</b> / (wa-ka-ri-ma-shi-ta) / 알겠습니다
      <ul><li>기억법: “와카리(알다) + 마시타(과거/정중)” 느낌만 잡기.</li></ul>
    </li>
    <li><b>わかりません</b> / (wa-ka-ri-ma-sen) / 모르겠습니다</li>
  </ol>

  <h2>4) 초간단 문장 패턴 2개</h2>
  <h3>패턴 A: “X 입니다”</h3>
  <pre><code>わたしは 〜 です。
(wa-ta-shi wa ~ de-su)
저는 ~입니다.</code></pre>
  <ul>
    <li>わたし(私): 나/저</li>
    <li>は: ‘와’라고 읽음(주제 표시)</li>
  </ul>

  <h3>패턴 B: “X 좋아해요”</h3>
  <pre><code>〜 が すきです。
(~ ga su-ki de-su)
~을(를) 좋아해요.</code></pre>

  <h2>5) 오늘의 복습(초기 세팅)</h2>
  <ul>
    <li><b>내일(D+1)</b>: 오늘 표현 10개를 “뜻 → 일본어”로 역방향 테스트</li>
    <li><b>3일 후(D+3)</b>: 패턴 A/B를 빈칸 채우기로 10문장 만들기</li>
    <li><b>7일 후(D+7)</b>: 표현 10개 + 패턴 2개로 1분 자기소개 만들기</li>
  </ul>
