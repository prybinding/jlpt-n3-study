# JLPT N3 Daily Pages — Curriculum Roadmap (v1)

원칙: **노베이스 기준**으로, 매일 한 페이지를 그대로 따라가면 누적되게 설계한다.

## Daily Page 구성(고정)

1) **단어/표현 30개**
- 표기(한자/가나)
- 가나
- 로마자(발음)
- 뜻(한국어)
- 기억법(연상/어원/형태/소리 고리)

2) **문법 2~4개**
- 패턴 / 뜻 / 포인트
- 예문 3~6개(가나/발음/뜻)

3) **복습(SRS)**
- D+1, D+3, D+7, D+14
- 원칙: **뜻→일본어(역방향)** 먼저 하고 확인

4) (추가 예정) **독해/청해**
- 초반 2주: 매우 짧은 독해(2~5문장)
- 이후: N4→N3 난이도로 점진 상승

---

## 단계 설계(대략)

### Phase 0 (Day 1~7): 생존 기초 + 학습 습관
- 가나(히라/가타) + 발음 규칙(장음/촉음/탁음)
- 인사/기본동사/핵심 조사 맛보기
- 문법: です/ます/ください/すき/あります・います(기초)

### Phase 1 (Day 8~21): N5 핵심 골격
- 동사 ます형/사전형 감각
- て형을 중심으로 문장 확장(요청/진행/허가/금지 등)
- 시간/숫자/요일/빈도 표현

### Phase 2 (Day 22~45): N4로 확장
- 형용사/형용동사 활용
- 가능형/의지형/수동/사역 개념의 초입
- 비교/이유/경험/의도
- 독해: 짧은 안내문/메모

### Phase 3 (Day 46~90): N3 본격 진입
- N3 핵심 문법(〜ように, 〜ために, 〜ことにする, 〜てしまう, 〜そうだ, 〜らしい, 〜わけだ 등)
- 어휘: 추상어/감정/업무/일상사
- 독해: 짧은 칼럼/설명문

## 데이터 운영 방식(A안)
- `data/plan/vocab.json`: vocab 풀(계속 확장)
- `data/plan/grammar.json`: 문법 풀(계속 확장)
- `scripts/render-day.mjs`: day N에 대해
  - vocab 30개를 순서대로 할당
  - 문법은 day-band(단계)에 맞는 항목을 선택
  - SRS는 이전 day들의 vocab을 끌어와 자동 구성

## 품질 기준
- 초반(1~14일) 기준: **초과 친절** (가나/발음/뜻/주의점/기억법)
- 중반 이후: 기억법은 "아무거나"가 아니라 **맥락/연상 고리**를 최소 1개는 제공
- 애매한 번역/뉘앙스는 (짧게) 주석으로 처리
