  <h2>2) 오늘의 문법 (2개)</h2>
  <div class="card">
    <h3>〜をください (~ 주세요)</h3>
    <p class="note"><b>패턴</b>: <code>N を ください</code><br>
    <b>읽기(로마자)</b>: ~ o ku-da-sa-i<br>
    <b>뜻</b>: N을 주세요(정중)</p>

    <h4>포인트</h4>
    <ul>
      <li>を는 목적어 표시. 발음은 '오'</li>
      <li>주문/요청에 매우 자주</li>
    </ul>

    <h4>예문</h4>
    <ol>
    <li>
      <b>みずを ください。</b><br>
      <span class="note">가나: <b>みずを ください。</b> · 발음(로마자): <b>mi-zu o ku-da-sa-i</b> · 뜻: <b>물을 주세요.</b></span>
    </li>
    <li>
      <b>これを ください。</b><br>
      <span class="note">가나: <b>これを ください。</b> · 발음(로마자): <b>ko-re o ku-da-sa-i</b> · 뜻: <b>이거 주세요.</b></span>
    </li>
    </ol>
  </div>
  <div class="card">
    <h3>〜ます / 〜ません (동사의 정중형 / 부정)</h3>
    <p class="note"><b>패턴</b>: <code>V ます / V ません</code><br>
    <b>읽기(로마자)</b>: ~ ma-su / ~ ma-sen<br>
    <b>뜻</b>: 합니다 / 하지 않습니다</p>

    <h4>포인트</h4>
    <ul>
      <li>초기에는 '사전형(たべる)'과 'ます형(たべます)'를 같이 익히면 좋음</li>
    </ul>

    <h4>예문</h4>
    <ol>
    <li>
      <b>たべます。</b><br>
      <span class="note">가나: <b>たべます。</b> · 발음(로마자): <b>ta-be-ma-su</b> · 뜻: <b>먹습니다.</b></span>
    </li>
    <li>
      <b>のみません。</b><br>
      <span class="note">가나: <b>のみません。</b> · 발음(로마자): <b>no-mi-ma-sen</b> · 뜻: <b>마시지 않습니다.</b></span>
    </li>
    </ol>
  </div>

  <h2>3) 복습 (SRS)</h2>
  <p class="note">원칙: ‘뜻 → 일본어’ 역방향으로 먼저 떠올리고, 그 다음에 확인.</p>
  <ol>
    <li><b>D+1</b> (Day 001)
      <ul>
        <li><b>こんにちは</b> (こんにちは) — 안녕하세요(낮)</li>
        <li><b>おはよう</b> (おはよう) — 좋은 아침/안녕(아침)</li>
        <li><b>こんばんは</b> (こんばんは) — 안녕하세요(저녁)</li>
        <li><b>ありがとう</b> (ありがとう) — 고마워요</li>
        <li><b>すみません</b> (すみません) — 죄송합니다/실례합니다/저기요</li>
        <li><b>はい</b> (はい) — 네</li>
        <li><b>いいえ</b> (いいえ) — 아니요</li>
        <li><b>お願いします</b> (おねがいします) — 부탁합니다/주세요</li>
        <li><b>わかりました</b> (わかりました) — 알겠습니다</li>
        <li><b>わかりません</b> (わかりません) — 모르겠습니다</li>
      </ul>
    </li>
  </ol>
